Учебные материалы

Введение в программирование

Мы живём в мире, где нас окружают компьютеры, смартфоны и другие цифровые устройства. Они выполняют команды, помогают в работе, развлечениях, науке и бизнесе. Но как они понимают нас? Как устройство узнаёт, что именно мы от него хотим?

Как компьютеры «понимают» нас?

На самом деле всё довольно просто. Любой компьютер или телефон работает по заранее заданному набору инструкций — шагов, которые устройство выполняет в строгом порядке. Эти инструкции говорят устройству, что делать: от простых операций вроде сложения чисел до сложных алгоритмов искусственного интеллекта.

Но здесь возникает важный вопрос: на каком языке передавать эти инструкции?

Машинный код – язык, понятный компьютеру

Компьютеры «разговаривают» на языке машинного кода. Это набор инструкций, закодированных в виде чисел, которые понимает процессор. Машинный код пишется в двоичном формате, состоящем из нулей и единиц.

Пример машинного кода:

10101100 00001111 11001010 00000001

Однако машинный код и двоичный код — это не одно и то же. Двоичный код — это всего лишь система счисления, представляющая информацию (текст, числа, команды) в виде 0 и 1. А машинный код — это конкретные команды, которые выполняет процессор, записанные в двоичной форме.

Проблема в том, что писать и понимать такой код сложно. Поэтому придумали языки программирования.

Языки программирования – посредник между человеком и машиной

Языки программирования позволяют писать команды в привычной для человека форме, а затем специальные программы (компиляторы и интерпретаторы) переводят их в машинный код.

Например, вместо машинного кода мы можем написать на языке Python:

				
					print("Привет, мир!")

И компьютер, с помощью интерпретатора Python, поймёт, что нужно вывести текст на экран.

Языки программирования можно сравнить с переводчиком:

  • Мы говорим на человеческом языке (Python, Java, C++).
  • Программа-переводчик (компилятор или интерпретатор) преобразует этот код в машинные команды.
  • Компьютер выполняет эти команды и выдаёт результат.

Где используется программирование?

Программирование применяется в самых разных сферах, и его значение постоянно растёт. Вот лишь некоторые области:

  • Разработка программ и мобильных приложений – создание программ для Windows, macOS, Android, iOS.
  • Веб-разработка – сайты, интернет-магазины, социальные сети.
     
  • Игровая индустрия – создание компьютерных игр (Unity, Unreal Engine).

  • Машинное обучение и искусственный интеллект – чат-боты, голосовые помощники, распознавание лиц.
  • Автоматизация процессов – написание скриптов для офисной работы, заводов, домашних устройств.
  • Разработка встроенных систем – программирование микроконтроллеров в роботах, бытовой технике, автомобилях.

Что такое программирование?

Программирование Это процесс написания инструкций на языке программирования, которые компьютер выполняет для решения определённой задачи.

Это может быть простая программа, которая выводит текст на экран, или сложная система, управляющая работой целого предприятия.

Кто такой программист?

Программист Это специалист, который создаёт программы с помощью языков программирования.

Программисты:

  • Анализируют задачи и придумывают решения.
  • Пишут код и тестируют его.
  • Оптимизируют программы, делая их быстрее и удобнее.
  • Работают с алгоритмами и структурами данных.

Сколько направлений, столько и языков программирования

В зависимости от области применения используются разные языки программирования:

  • Для веб-разработки – JavaScript, PHP, Python.
  • Для создания игр – C++, C# (Unity).
  • Для машинного обучения – Python, R.
  • Для системного программирования – C, Rust.

Хотя языки программирования имеют отличия, у них есть общие принципы:

  • Работа с переменными (хранение данных).
  • Условные конструкции (если …, то …).
  • Циклы (повторение действий).
  • Функции (разбиение кода на удобные блоки).

Первая программа: «Hello, World!»

Традиционно изучение любого языка программирования начинается с написания программы, которая просто выводит на экран фразу «Hello, World!» («Привет, мир!»).

История «Hello, World!»

Впервые эта программа появилась в 1978 году в книге «The C Programming Language» авторов Брайана Кернигана и Денниса Ритчи. Они использовали её в качестве самого простого примера для изучения языка C. С тех пор традиция распространилась на все языки программирования.

Примеры «Hello, World!» на разных языках:

 C

				
					#include <stdio.h>
int main() {
    printf("Hello, World!");
    return 0;
}

Java

				
					public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

JavaScript

				
					console.log("Hello, World!");

Python

				
					print("Hello, World!")

Переход к изучению Python

Теперь, когда мы разобрались с основами программирования и настроили среду разработки, давайте познакомимся с одним из самых популярных языков программирования — Python.

Python Это мощный и универсальный язык, который широко используется в различных сферах IT. Его популярность обусловлена простотой синтаксиса, удобочитаемостью кода и огромным сообществом разработчиков.

Где используется Python?

Python применяется в самых разных областях:

  • Веб-разработка – создание сайтов и веб-приложений.
  • Анализ данных и машинное обучение – обработка больших данных, нейросети, искусственный интеллект.
  • Автоматизация задач – написание скриптов, которые выполняют рутинные процессы.
  • Кибербезопасность – создание инструментов для тестирования безопасности.
    Разработка игр – создание игр на основе Pygame.
     
  • Встраиваемые системы и IoT – программирование микроконтроллеров.

Python активно используется крупнейшими компаниями, такими как Google, Facebook, NASA, Netflix и многими другими.

Интерпретируемый язык программирования

Python относится к интерпретируемым языкам программирования.

Что это значит?

  • В отличие от компилируемых языков (например, C или Java), Python не требует предварительной компиляции в машинный код.
  • Вместо этого специальная программа – интерпретатор Python – выполняет код построчно, переводя его в машинные инструкции на лету.

Это можно сравнить с тем, как человек читает книгу.

  • Если код компилируемый – это значит, что сначала вся книга переводится целиком, а затем мы можем её прочитать.
  • Если код интерпретируемый (как в Python), то компьютер читает и выполняет код построчно, так же как человек читает книгу по одной строке за раз.

Пример работы интерпретатора:

				
					# Строка 1
print("Привет, мир!")
# Строка 2
print("Добро пожаловать в Python!")

Интерпретатор как переводчик, может переводить лишь те команды, которые он знает сам, а те, что не знает воспринимает за ошибку. Но что делать, если необходимо написать дополнительную информацию о написанном коде? Для такого рода задач в Python есть «комментарии».

Комментарии — это участок в программе, который интерпретатор не считывает и не переводит. Для того чтобы написать комментарий необходимо перед текстом поставить #. Комментарий не обязательно должен начинаться с начала строки, их можно писать в любой момент строки, главное помнить, что справа от знака # всё считается комментарием и работать не будет.

Ввод и вывод данных в Python

Теперь же, давайте сами напишем нашу первую программу. Это программой станет, ранее упомянутая, программа “Hello World”, которая на языке программирования Python выглядит следующим образом:

				
					print(“Hello World”)
#Программа выведет на экран: “Hello World”

Программа вывела в консоль сообщение которое мы указали внутри скобок и кавычек, давайте же разберём программу более подробно.

print — это команда, которая даёт компьютеру инструкцию: “Выведи что-то на экран”.

«Hello World» — в данном контексте является текстом, который мы и выводим. В работе с текстом важно записывать его в кавычки, чтобы Python понял, что это текст а не команда.

Попробуем изменить программу, например сделаем программу русскоговорящей:

				
					print("Привет, мир!")
#Программа выведет на экран: “Привет, мир!”

print у нас остался тот же, поэтому само действие программы не изменилось, однако изменился результат, вывод стал на русском языке.

Давайте ещё изменим программу и выведем сразу несколько строчек:

				
					print("Привет, мир!")  
print("Добро пожаловать в Python!")  
print("Это моя первая программа!")

Результатом этой программы станет построчное выполнение команды print, которая выводит текст внутри скобок. Следовательно в окне консоли мы увидим следующий результат:

				
					Привет, мир!  
Добро пожаловать в Python!  
Это моя первая программа!

Сейчас же, сделаем нашу программу интерактивной, добавив команду input().

input() — команда, которая даёт возможность пользователю(нам) ввести данные в консоль. Это может быть ответ на вопрос или сообщение, которое надо вывести или передать. Также как и команда print(), может выводить сообщения, если также как и с командой print(), указать текст внутри скобок.

				
					name = input("Как тебя зовут? ")  
print("Привет, " + name + "!")

Результатом работы данной программы является следующий ряд событий:

1.Программа запрашивает пользователя — выводит информацию на экран, при помощи команды input()

2.Пользователь вводит свой ответ на вопрос как его зовут — вводит в программу данные, благодаря команде input()

3.Программа при помощи команды print() выводит сообщение, состоящее из трёх частей, одна из которых является нашим ответом.

Команды print() и input(), в комбинации друг с другом, позволяют полноценно взаимодействовать с компьютером: print — выводит текст на экран; input — позволяет вводить данные.
