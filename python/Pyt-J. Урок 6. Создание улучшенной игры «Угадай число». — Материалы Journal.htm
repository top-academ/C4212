Pyt-J. Урок 6. Создание улучшенной игры «Угадай число».

Учебные материалы

На прошлых уроках мы уже создавали простую версию игры «Угадай число». Она помогла нам освоить базовые конструкции языка Python: работу с условиями, циклами и вводом данных от пользователя. Однако сегодня мы сделаем шаг дальше! Теперь наша задача — не просто повторить старую игру, а превратить её в полноценное приложение с несколькими режимами, настройкой сложности и красивым оформлением.
 В этом уроке мы научимся:

  • Организовывать структуру программы через главное меню.
  • Создавать разные режимы игры: одиночный (против компьютера) и парный (игрок против игрока).
  • Управлять сложностью игры с помощью ограничения числа попыток.
  • Продумывать пользовательский интерфейс, чтобы программа была удобной и понятной.

Давайте начнём с подготовки нашего проекта!

Подготовка и подключение библиотеки

Для работы игры нам понадобится модуль random, который позволяет компьютеру генерировать случайные числа.

Сегодня мы только познакомились с библиотекой random.
А на следующем уроке мы начнём подробнее разбирать библиотеки.

Библиотека random Это ключевой элемент для одиночного режима, где компьютер загадывает число.

Однако мы не можем использовать его в обычном Python, с чем нам и поможет одноимённая библиотека

				
					import random

import random — команда, которая подключает в нашу программу модуль random.

Модуль random позволяет использовать функции для генерации случайных чисел.

Без подключения этой библиотеки мы не сможем, например, загадать случайное число для игры против компьютера.

Важно! Подключение происходит один раз в начале программы.

Главное меню игры

Главное меню представляет собой центральный узел нашей программы, выполняющий несколько важных функций.

Это не просто список вариантов выбора, а сложная система взаимодействия с пользователем, которая требует тщательной проработки каждого элемента.

Давайте рассмотрим его компоненты и их назначение более подробно.

				
					
print("=" * 45)
print("||" + " " * 10 + "УГАДАЙ ЧИСЛО 2.0" + " " * 10 + "||")
print("=" * 45)
print("\nДобро пожаловать в улучшенную версию классической игры!")
  • print(«=» * 45) — печатает 45 символов «=» подряд для создания рамки.

  • print(«||» + » » * 10 + «УГАДАЙ ЧИСЛО 2.0″ + » » * 10 + «||») — делает красивый заголовок игры внутри рамки.

  • print(«\nДобро пожаловать…») — выводит приветственное сообщение на новую строку.

Такой подход делает интерфейс дружелюбным и понятным пользователю.

Код основных пунктов меню:

				
					print("\nДоступные режимы игры:")
print("-" * 30)
print("1. Одиночная игра против компьютера")
print("   (Компьютер загадывает число от 1 до 100)")
print("2. Парная игра с другом")
print("   (Один игрок загадывает, второй угадывает)")
print("3. Выход из программы")
print("-" * 30)
  • Мы выводим доступные режимы игры.

  • Описание под каждым пунктом помогает игроку сразу понять, что делает каждый режим.

  • Линии из «-» служат для аккуратного визуального отделения меню.

Код для выбора пункта меню:

				
					
while True:
    choice = input("\nВведите номер выбранного режима (1-3): ")
    
    if choice != "1" and choice != "2" and choice != "3":
        print("\nОшибка: нужно ввести 1, 2 или 3!")
        print("Пожалуйста, попробуйте еще раз.")
        continue
    
    break

Пояснение:

  • while True — бесконечный цикл, который будет повторяться, пока пользователь не введет корректное значение.
  • choice = input(…) — запрашиваем ввод у пользователя.
  • if choice != «1» and choice != «2» and choice != «3»: — проверяем, ввёл ли он допустимый вариант. Если нет — выдаём ошибку и возвращаемся к началу цикла.
  • break — если всё хорошо, выходим из цикла.

Одиночный режим игры

Начало одиночного режима:

				
					
if choice == "1":
    print("\n=== Одиночный режим ===")
    secret_number = random.randint(1, 100)
    attempts = 0
    max_attempts = None
  • secret_number = random.randint(1, 100) — загадываем случайное число от 1 до 100.

  • attempts = 0 — создаём счётчик попыток, который будет увеличиваться после каждого угадывания.

  • max_attempts = None — пока лимит на количество попыток не установлен.

Установка лимита попыток:

				
					
    limit_choice = input("Установить лимит попыток? (да/нет): ").lower()
    if limit_choice == "да":
        max_attempts = int(input("Введите максимальное количество попыток: "))
  • Спрашиваем у игрока, хочет ли он ограничить количество попыток.

  • Если да, спрашиваем, сколько именно попыток будет разрешено.

Основной игровой цикл:

				
					
print("Компьютер загадал число от 1 до 100. Попробуйте угадать!")
while True:
    guess = int(input("Ваша догадка: "))
    attempts += 1
  • Выводим сообщение о начале угадывания.

  • Пользователь вводит свою догадку, а программа увеличивает счётчик попыток.

Проверка ответа:

				
					
while True:
        guess = int(input("Ваша догадка: "))
        attempts += 1

        if guess < secret_number:
            print("Загаданное число БОЛЬШЕ!")
        elif guess > secret_number:
            print("Загаданное число МЕНЬШЕ!")
        else:
            print(f"Поздравляем! Вы угадали за {attempts} попыток!")
            break

Сравниваем введённое число с загаданным:

  • Если догадка меньше — говорим «больше».

  • Если больше — говорим «меньше».

  • Если равно — поздравляем и выходим из цикла.

Проверка ограничения попыток:

				
					
if max_attempts and attempts >= max_attempts:
            print(f"Попытки закончились! Загаданное число было {secret_number}.")
            break
  • Если установлен лимит попыток, проверяем, не израсходовал ли игрок все попытки.

  • Если да, сообщаем об окончании игры и показываем правильный ответ.

Парный режим игры

Инициализация парного режима:

				
					
elif choice == "2":
    print("\n=== Парный режим ===")
    print("Игрок 1 загадывает число, Игрок 2 угадывает")
    secret_number = int(input("Игрок 1, введите число от 1 до 100: "))
    attempts = 0
    max_attempts = None
  • Теперь число загадывает Игрок 1.
  • Все остальные переменные такие же, как и в одиночном режиме.
				
					 
 limit_choice = input("Установить лимит попыток? (да/нет): ").lower()
    if limit_choice == "да":
        max_attempts = int(input("Введите максимальное количество попыток: "))

Основной игровой цикл использует ту же структуру while True, но с адаптированными сообщениями для парного режима. Каждая итерация включает:

				
					
print("Игрок 2, начинайте угадывать!")
    
    while True:
        guess = int(input("Ваша догадка: "))
        attempts += 1
        
        if guess < secret_number:
            print("Загаданное число БОЛЬШЕ!")
        elif guess > secret_number:
            print("Загаданное число МЕНЬШЕ!")
        else:
            print(f"Игрок 2 угадал число за {attempts} попыток!")
            break

Особое внимание уделено ясности сообщений. Все подсказки и результаты формулируются максимально четко, чтобы оба игрока одинаково понимали состояние игры. Например, сообщение о победе явно указывает, что число угадал «Игрок 2», а не абстрактное «вы», как в одиночном режиме.   

				
					
if max_attempts and attempts >= max_attempts:
            print(f"Попытки закончились! Загаданное число было {secret_number}.")
            break

Валидация ввода реализована на базовом уровне — преобразование в целое число, но, как и в одиночном режиме, здесь остается возможность для улучшений.

Завершение программы

Прописываем условия выхода из программы:

				
					
elif choice == "3":
    print("Спасибо за игру! До свидания!")
  • Если пользователь выбрал «3», мы вежливо прощаемся и программа завершается.

  • Это важно для формирования позитивного пользовательского опыта.

Полный код программы:

				
					
import random

print("=" * 45)
print("||" + " " * 10 + "УГАДАЙ ЧИСЛО 2.0" + " " * 10 + "||")
print("=" * 45)
print("\nДобро пожаловать в улучшенную версию классической игры!")

print("\nДоступные режимы игры:")
print("-" * 30)
print("1. Одиночная игра против компьютера")
print("   (Компьютер загадывает число от 1 до 100)")
print("2. Парная игра с другом")
print("   (Один игрок загадывает, второй угадывает)")
print("3. Выход из программы")
print("-" * 30)

while True:
    choice = input("\nВведите номер выбранного режима (1-3): ")
    
    if choice != "1" and choice != "2" and choice != "3":
        print("\nОшибка: нужно ввести 1, 2 или 3!")
        print("Пожалуйста, попробуйте еще раз.")
        continue

    if choice == "1":
        print("\n=== Одиночный режим ===")
        secret_number = random.randint(1, 100)
        attempts = 0
        max_attempts = None

        limit_choice = input("Установить лимит попыток? (да/нет): ").lower()
        if limit_choice == "да":
            max_attempts = int(input("Введите максимальное количество попыток: "))

        print("Компьютер загадал число от 1 до 100. Попробуйте угадать!")
        while True:
            guess = int(input("Ваша догадка: "))
            attempts += 1

            if guess < secret_number:
                print("Загаданное число БОЛЬШЕ!")
            elif guess > secret_number:
                print("Загаданное число МЕНЬШЕ!")
            else:
                print(f"Поздравляем! Вы угадали за {attempts} попыток!")
                break
            if max_attempts and attempts >= max_attempts:
                print(f"Попытки закончились! Загаданное число было {secret_number}.")
                break

    elif choice == "2":
        print("\n=== Парный режим ===")
        print("Игрок 1 загадывает число, Игрок 2 угадывает")
        secret_number = int(input("Игрок 1, введите число от 1 до 100: "))
        attempts = 0
        max_attempts = None

        limit_choice = input("Установить лимит попыток? (да/нет): ").lower()
        if limit_choice == "да":
            max_attempts = int(input("Введите максимальное количество попыток: "))

        print("Игрок 2, начинайте угадывать!")
    
        while True:
            guess = int(input("Ваша догадка: "))
            attempts += 1
        
            if guess < secret_number:
                print("Загаданное число БОЛЬШЕ!")
            elif guess > secret_number:
                print("Загаданное число МЕНЬШЕ!")
            else:
                print(f"Игрок 2 угадал число за {attempts} попыток!")
                break
            if max_attempts and attempts >= max_attempts:
                print(f"Попытки закончились! Загаданное число было {secret_number}.")
                break

    elif choice == "3":
        print("Спасибо за игру! До свидания!")

На этом мы закончили разработку продвинутой игры «Угадай число»
