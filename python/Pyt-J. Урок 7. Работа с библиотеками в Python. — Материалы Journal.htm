Pyt-J. Урок 7. Работа с библиотеками в Python.

Учебные материалы

Что такое библиотеки в Python?

На прошлом занятии, мы разработали продвинутую игру «Угадай число». Для реализации одиночного режима необходимо было использовать библиотеку random.

Но что такое библиотека в Python? Как ею пользоваться? Для чего она нужна?

Библиотеки в Python Это наборы функций, классов и модулей, которые упрощают решение различных задач. Вместо того чтобы каждый раз писать код для распространенных операций, можно подключить библиотеку, которая уже включает готовые решения. Это позволяет ускорить разработку и сделать код более читаемым.

Например, для выполнения математических операций мы можем использовать встроенную библиотеку math, а для работы с случайными числами — библиотеку random. Однако Python предоставляет множество других библиотек для различных задач, включая графику, веб-разработку, работу с данными и многое другое.

Зачем использовать библиотеки?

Когда вы подключаете библиотеку в Python, вы получаете доступ к её методам и классам, которые значительно упрощают выполнение специфических задач. Это сэкономит вам время и силы, а также повысит читаемость вашего кода.

Для подключения библиотеки в Python используется конструкция import:

				
					import название_библиотеки

Затем вы можете использовать все функции и классы, доступные в этой библиотеке. Рассмотрим несколько примеров.

Основные библиотеки Python

1. Библиотека math

Библиотека math включает множество полезных математических функций, которые пригодятся при решении задач, связанных с числами. Рассмотрим некоторые из них.

Примеры:

				
					
import math

# Квадратный корень числа
print(math.sqrt(16))  # 4.0

# Вычисление факториала
print(math.factorial(5))  # 120

# Синус угла (в радианах)
print(math.sin(math.pi / 2))  # 1.0

# Возведение в степень
print(math.pow(2, 3))  # 8.0

Здесь:

  • sqrt(16) вычисляет квадратный корень из 16.
  • factorial(5) возвращает факториал числа 5.
  • sin(math.pi / 2) вычисляет синус угла, равного 90 градусам (в радианах).
  • pow(2, 3) возводит число 2 в степень 3.

2. Библиотека random

Библиотека random используется для генерации случайных чисел. Это полезно, когда нужно случайным образом выбрать элементы, например, для игр, тестов или случайных событий.

Примеры:

				
					
import random

# Случайное целое число от 1 до 10
print(random.randint(1, 10))

# Случайный элемент из списка (списки мы разберём чуть позже)
fruits = ["яблоко", "банан", "вишня"]
print(random.choice(fruits))

# Перемешивание списка
random.shuffle(fruits)
print(fruits)

Здесь:

  • randint(1, 10) генерирует случайное целое число от 1 до 10.
  • choice(fruits) выбирает случайный элемент из списка.
  • shuffle(fruits) перемешивает элементы списка.

Способы импорта библиотек в Python

В Python существует несколько способов импорта библиотек и модулей. Рассмотрим основные методы.

  • Обычный импорт

Самый простой способ импортировать библиотеку — использовать команду import:

				
					import math
import random

Этот способ позволяет получить доступ ко всем функциям и классам библиотеки, используя имя библиотеки в качестве префикса:

				
					math.sqrt(16)
random.randint(1, 10)
  • Импорт с псевдонимом

Если название библиотеки слишком длинное или если вы хотите использовать короткое имя для удобства, вы можете задать псевдоним (alias):

				
					import math as m
import random as r

Теперь вы можете использовать псевдонимы для обращения к библиотекам:

				
					m.sqrt(16)
r.randint(1, 10)
  • Импорт отдельных функций или классов

Если вам нужно только несколько функций или классов из библиотеки, можно импортировать их напрямую, без использования префикса с именем библиотеки. Это особенно полезно, если библиотека содержит множество функций, а вам нужно только несколько:

				
					from math import sqrt, pow
from random import randint

Теперь можно использовать функции напрямую:

				
					sqrt(16)
pow(2, 3)
randint(1, 10)
  • Импорт всего содержимого библиотеки

В случае, если вам нужно использовать все функции и классы из библиотеки, можно импортировать её полностью:

				
					from math import *
from random import *

Этот способ не рекомендуется использовать в больших проектах, так как он может привести к конфликту имен, если в разных библиотеках есть одноимённые функции. Лучше использовать первый или второй способ, чтобы избежать таких проблем.

  • Импорт модулей из пакетов

Python поддерживает иерархию пакетов, то есть библиотеки могут содержать несколько модулей. Чтобы импортировать определённый модуль из пакета, нужно указать путь к нему:

				
					from package_name.module_name import function_name

Пример:

				
					from datetime import datetime

В этом примере мы импортируем класс datetime из модуля datetime пакета datetime.

3. Библиотека Turtle

Теперь давайте сосредоточимся на библиотеке turtle, которая является одной из самых интересных и визуально привлекательных. С помощью этой библиотеки можно рисовать на экране, управляя «черепашкой», которая оставляет за собой след.

Подключаем библиотеку

Для того чтобы начать работать с библиотекой turtle, нам нужно её подключить. Это делается с помощью команды import:

				
					import turtle

Теперь мы можем использовать все возможности библиотеки, такие как управление черепашкой, рисование и настройка внешнего вида.

Настройка внешнего вида черепашки

Первым делом давайте настроим нашу черепашку. Для этого можно изменить её форму, цвет и скорость.

Изменение формы черепашки

Черепашка может выглядеть по-разному. Для этого используем команду shape(). Вот возможные формы:

  • «arrow» — стрела

  • «turtle» — черепашка (по умолчанию)

  • «circle» — круг

  • «square» — квадрат

  • «triangle» — треугольник

  • «classic» — классический вид

Пример:

				
					turtle.shape("turtle")  # Устанавливаем форму черепашки

Изменение цвета

Вы можете изменить цвет, которым черепашка будет рисовать линии, с помощью команды color():

				
					turtle.color("green")  # Устанавливаем цвет линий

Изменение скорости

Черепашка может двигаться с разной скоростью от 1 (медленно) до 10 (быстро):

				
					turtle.speed(5)  # Устанавливаем скорость движения

Двигаем черепашку и рисуем

Теперь давайте научим черепашку двигаться и рисовать. Для этого используются несколько основных команд:

  • forward(distance) — движение вперёд на указанное количество пикселей.

  • backward(distance) — движение назад на указанное количество пикселей.

  • left(angle) — поворот на указанный угол влево.

  • right(angle) — поворот на указанный угол вправо.

Давайте нарисуем квадрат с помощью этих команд. Для этого черепашка будет двигаться вперёд на 30 пикселей, затем поворачивать налево на 90 градусов, и так повторяем четыре раза.

				
					import turtle

# Настроим внешний вид черепашки
turtle.shape("turtle")  # Устанавливаем форму черепашки
turtle.color("green")   # Устанавливаем цвет линий
turtle.speed(5)         # Устанавливаем скорость

# Рисуем квадрат
turtle.forward(30)
turtle.left(90)
turtle.forward(30)
turtle.left(90)
turtle.forward(30)
turtle.left(90)
turtle.forward(30)

turtle.done()  # Завершаем работу

Использование циклов для удобства

Программа работы черепашки идеальная демонстрация повторения одного и того же действия, и вместо того, чтобы повторять эти действия по нескольку раз, мы можем использовать циклы. Изменим программу для рисования квадрата, с применением циклов. Т.к. мы знаем сколько раз надо выполнить одно и то же действие, то правильней использовать цикл for:

				
					import turtle

turtle.shape("turtle")  
turtle.color("green")
turtle.speed(5)         

for i in range(4):
    turtle.forward(30)
    turtle.left(90)

turtle.done()

Черепашка выполняет ту же программу, только вместо 4-х кратного повторения её действий, она идёт по нашему циклу.

Использование переменных для удобства

Если вам нужно быстро изменять размер сторон квадрата, вместо того чтобы менять число в каждой строке, можно использовать переменные. Например, давайте объявим переменную size, которая будет хранить размер стороны квадрата:

				
					import turtle

turtle.shape("turtle")
turtle.color("green")
turtle.speed(5)

# Устанавливаем размер квадрата с помощью переменной
size = 50

for i in range(4):
    turtle.forward(size)
    turtle.left(90)

turtle.done()

Теперь, чтобы изменить размер квадрата, достаточно изменить значение переменной size — и все стороны квадрата сразу изменятся.

Рисуем окружности

Черепашка может рисовать не только прямые линии, но и окружности. Для этого используется метод circle(), в который нужно передать радиус окружности:

				
					import turtle

turtle.shape("turtle")
turtle.color("blue")
turtle.speed(5)

# Рисуем окружность с радиусом 50 пикселей
turtle.circle(50)

turtle.done()

Перемещение без рисования

Иногда бывает нужно перемещать черепашку, но не рисовать при этом линию. Для этого используем команды penup() и pendown():

  • penup() — поднимает перо, и черепашка перестаёт рисовать.

  • pendown() — опускает перо, и черепашка снова начинает рисовать.

Пример:

				
					import turtle

turtle.shape("turtle")
turtle.color("red")
turtle.speed(5)

# Двигаем черепашку без рисования
turtle.penup()  # Поднимаем перо
turtle.forward(100)  # Двигаем черепашку вперёд
turtle.pendown()  # Опускаем перо

for i in range(4):
    turtle.forward(50)
    turtle.left(90)

turtle.done()
